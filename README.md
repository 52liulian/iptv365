## 📋 项目简介

IPTV365 是一个自用的 IPTV 直播源收集、整理和更新的项目，通过自动化脚本每日更新直播源。

## 📊 源说明

| 类别 | 文件名 | 更新频率 | 说明 |
|------|-------|----------|------|
| 标准版 | live.txt | 每日 | 完整直播源列表（TXT 格式） |
| 标准版 | live.m3u | 每日 | 完整直播源列表（M3U 格式） |
| 精简版 | live_lite.txt | 每日 | 精选高质量直播源（TXT 格式） |
| 精简版 | live_lite.m3u | 每日 | 精选高质量直播源（M3U 格式） |
| 黑名单 | blacklist_auto.txt | 不定时 | 自动过滤的无效直播源 |
| 白名单 | whitelist_auto.txt | 不定时 | 高响应优质直播源 |
